Microsoft Antitrust Case Timeline

Pre-1995: Microsoft establishes and maintains a monopoly share (over 80%) in the PC operating system market
1994: Netscape Navigator is introduced and becomes the first widely used Internet browser
July 1995: Microsoft releases Internet Explorer 1.0
May 1995: Microsoft executives meet with Netscape and allegedly attempt to divide the browser market
August 1995: Microsoft releases Windows 95
Late 1995: Microsoft recognizes the threat posed by Netscape's browser and Sun's Java technologies
Early 1996: Microsoft begins offering Internet Explorer to Internet Access Providers (IAPs) for free
Spring 1996: Microsoft starts negotiations with Compaq to include Internet Explorer on Presario computers
May 1996: Microsoft allegedly threatens to terminate Compaq's Windows 95 license over Internet Explorer icon removal
June 1996: Compaq restores the MSN and Internet Explorer icons to the Presario desktop
Summer 1996: Microsoft introduces the Internet Explorer Access Kit (IEAK) for IAPs
July 1996: Microsoft enters into an agreement with AOL to make Internet Explorer the default browser
August 1996: Microsoft imposes restrictions on OEMs' ability to modify the Windows 95 boot-up sequence
Fall 1996: Microsoft introduces the Active Desktop and Online Services Folder in Windows
Late 1996: Microsoft integrates Internet Explorer more deeply into Windows operating system
1996-1997: Microsoft enters into restrictive agreements with Internet Content Providers (ICPs)
January 1997: Microsoft allegedly pressures Intel to stop developing platform-level software that could support Navigator
Mid-1997: Microsoft enters into agreements with ICPs for channel placement in Internet Explorer 4.0
September 1997: Microsoft releases Internet Explorer 4.0
October 1997: DOJ files a petition for contempt against Microsoft for alleged violation of 1995 consent decree
December 1997: Court issues preliminary injunction against Microsoft tying Internet Explorer to Windows 95
Early 1998: Microsoft's share of the browser market reaches approximately 50%, up from less than 5% in early 1996
April 1998: Microsoft modifies some of its agreements with ISPs, but not with Online Service Providers
May 1998: Court of Appeals stays the December 1997 preliminary injunction as it applies to Windows 98
June 1998: Anticipated release of Windows 98 with integrated Internet Explorer
